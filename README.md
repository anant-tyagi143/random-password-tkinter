# Random Password Generator (Tkinter)

## Overview
A simple desktop application built with Python Tkinter that generates random passwords and allows users to copy them to the clipboard.

## Features
- Generate an 8-character random password
- Uses letters and numbers
- Copy password to clipboard with one click
- Simple and clean GUI

## Requirements
- Python 3.x
- Tkinter (comes preinstalled with Python)
- No external libraries required

## How to Run
1. Save the script as:
   password_generator.py

2. Run the program:
   python password_generator.py

## How It Works
- Click **Generate Password** to create a new password.
- Click **Click to Copy** to copy the password.
- A short message confirms the copy action.

Simple project useful for learning Tkinter and basic Python GUI programming.
